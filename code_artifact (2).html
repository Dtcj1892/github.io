📱 手機離線 App (PWA) 部署助手與檔案下載 ▼

本系統已完全支援 PWA 漸進式網頁應用。要在手機上像獨立 App 一樣直接點擊開啟並 100% 離線運作（斷網仍會保留完整的 CSS 樣式、字型與防呆驗證），請依照以下指引快速完成：

  1. 點選下載按鈕，取得 PWA 所需的兩個核心設定檔：
  2. 將剛下載的 manifest.json 與 sw.js，放至與這個 index.html 同一個目錄（資料夾）下。
  3. 將整個資料夾上傳至您具有 HTTPS 的伺服器空間，或是任何免費的託管平台（例如：GitHub Pages、Netlify 或 Vercel）。
  4. 初次載入（快取建立）：首次在手機瀏覽器中開啟您的網址（此時系統會自動把 Tailwind 樣式及字型完整存進手機快取中）。
  5. 安裝 App：點擊瀏覽器功能列中的「新增至主畫面」（iOS 請按「分享 ➡️ 新增至主畫面」；Android 請點擊「安裝應用程式」），即可在桌面開啟並完全離線運作！
⚠️ 核心提示：PWA 技術要求必須在安全性協定（HTTPS 或本地 localhost）下才能運作。如果您直接在手機上雙擊開啟本機 html 檔 (file:// 協定)，瀏覽器會因為安全機制禁止啟用離線快取和安裝功能。
